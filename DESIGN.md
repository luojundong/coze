# DESIGN.md

## 气质与意象
专业可信的 SaaS 工具平台 — 干净的白底界面、克制的蓝灰主色、清晰的层次，如同一个整洁的现代办公桌面。

## 配色方案
- 主色：#2563EB (信任蓝) — 来自专业工具平台的可靠性意象
- 辅助色：#10B981 (成功绿) — 激活成功的视觉反馈
- 警告色：#F59E0B (注意黄) — 待激活/即将过期状态
- 危险色：#EF4444 (警示红) — 错误、拒绝
- 背景：#FFFFFF / #F9FAFB (浅灰底)
- 文字：#111827 (主) / #6B7280 (辅)

## 字体排版
- 中文优先：思源黑体 / 系统默认
- 英文：Inter
- 排版节奏：标题 20-24px 半粗、正文 14-16px 常规、辅助文字 12px

## 页面结构
- 登录/注册：居中卡片式，上方应用图标+名称
- 激活页：居中输入框 + 激活按钮
- Dashboard：左侧导航 + 右侧主内容区
  - 工作流列表卡片
  - 调用记录表格
  - 账户状态概览

## 设计禁忌
- 不要使用渐变背景
- 不要使用花哨动画
- 不要使用纯黑色文字
